My Work

Interactive Twitch Stream Moderation Bot | C# & API | 2024

This is an interactive Twitch Chat Bot that can be added to and edited to fit into any streamers chat. This currently can read and display messages from Twitch Chat directly inside of the console, and will respond to the streamer inputting chat commands into twitch chat directly using the TwitchLib.API. The currently functionality is limited to you, the channel's streamer, inputting commands like "!Game" which is case sensitive, which will display the response associated with the command.

Skills

  • Basic C# programming experience.
  • Basic networking like web requests which help in understanding how to send and receive data over the internet using HTTP requests and managing responses. Furthermore, I explored formats like JSON when interacting with the Twitch API, learning how to serialize and deserialize data.
  • Asynchronous execution to handle incoming chat messages and API requests concurrently, which is key in real-time applications.
  • Processing Commands like text input from users (in this case, Twitch chat commands like !Game) and trigger responses based on those inputs.

Web Scraper with CSV generation | C# & HTML | 2024

This program uses the HTML Agility Pack library to scrape weather information from Weather.com and CSVHelper to save the data into a CSV file for easy manipulation and viewing in a table format.

Skills

  • Data parsing by using the HTML Agility Pack which allows the program to navigate the HTML structure and locate specific elements, such as the temperature and weather condition, using XPath.
  • Practiced the fundamentals of string manipulation by extracting specific pieces of data from the raw HTML by trimming spaces, handling special characters, and parsing the content.
  • Basic web development knowledge by focusing on web scraping. The project gave basic exposure to web technologies such as HTTP, HTML, and CSS. This knowledge is important for understanding how websites are structured and delivered to users.
  • Exporting data to be used on other tools by storing data in CSV format, which is a practical skill since they are widely used in tools involving SQL development.

Plagiarism Checker | Python | 2023

The program uses cosine_similarity to find the differences and similarity between two text documents to find if there is any plagiarism. This will take two vectors which are made from the text documents raw data, and looks for the similarities between the two.

Skills

  • Feature extraction, which is a key concept in machine learning, where raw data must be transformed into numerical features that machine learning algorithms can process.
  • Enhanced my algorithmic thinking in Python so that I can transform raw text data into vectors and computing their similarities.
  • Text processing and natural language processing techniques.
  • Validating whether the program correctly detects plagiarism based on different levels of similarity which strengthens quality assurance and metric evaluation.

Tic Tac Toe | C# | 2023

This program is built in Visual Studio (Windows Forms) and features a simple TicTacToe game using Forms to create a visual in which a Player Vs CPU score based game is ran, the first player to get 3 “O”s or “X”s in a row wins. The CPU uses a random variable to determine what spots it will choose. Winning the game adds to the scoreboard displayed for either the player or the CPU. There is also a restart game option if the player chooses to immediately terminate the game with its stats and go again with a clean slate.

Skills

  • Using conditionals to check if a player or CPU has won the game, or if the game is a draw.
  • Learning how to use Windows Forms to create a simple GUI. This includes setting up buttons for each square on the Tic-Tac-Toe grid, text boxes for displaying scores, and labels for game instructions.
  • Implementing logic to check if a player or the CPU has won the game by getting 3 "X"s or "O"s in a row, column, or diagonal.
  • Get comfortable with accessing and modifying data in arrays since the game logic relies on indexing into the array to check or update the board.
  • Learning how to update and display scores based on game outcomes by incrementing a score variable and displaying it on the screen.

Pong Game | C# | 2023

This program is built in Visual Studio (Windows Forms) and features the classic game Pong utilizing several moving parts to make a Player Vs CPU score based game where in order to win, either side must be the first to score a set amount of points. The creation of this game involves heavy reliance on arrays, random variables, incremental changes in speed upon the XY axis, and IF ELSE statements to create a visual of a ball being hit between a player controlling a bar using the arrows keys and a CPU.

Skills

  • Game difficulty being balanced through the CPU's behavior which is adjusted based on the game state, such as the score or random chance.
  • Implementing the ball’s movement, especially how it bounces off walls or either player, introduces simple physics concepts, such as velocity, angle of reflection, and collision response.
  • Managing the game loop and controlling how quickly the screen updates or the ball moves involves using timers, which is important for game performance and user experience.
  • Event handling through user inputs being read on the arrow key usage.
  • Learning how to create a simple game loop which continuously updates the game state and renders the graphics (movement, collisions, etc).
  • Basic programming concepts like loops since the game requires constantly updating the positions of objects (ball, paddle) and checking for win conditions.
